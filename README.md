# java-racingcar
자동차 경주 게임 미션 저장소

### 요구사항 정리
- [x] 자동차를 생성하는 기능
- [ ] N대의 자동차를 입력받는 기능
    - [ ] 입력 형식(쉼표)을 잘 지켰는지 확인하는 기능 
    - [ ] 자동차 이름 길이(5 이하) 확인하는 기능
- [ ] 이동횟수 입력 기능
- [ ] 0 ~ 9 사이의 값 랜덤 값 생성
    - [ ] 0 ~ 3은 정지, 4 ~ 9는 전진 
- [ ] 레이싱 과정을 횟수별로 출력
- [ ] 우승자 결정하는 기능
    - [ ] 공동 우승자인 경우 콤마로 구분해서 출력
